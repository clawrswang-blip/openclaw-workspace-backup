## 📋 操作规程

### 添加新问题（任何时候）
```
格式：直接在上方表格追加一行
触发条件：
  - Rishon 明确表示不满
  - 自己意识到持续性问题
  - 指标低于阈值连续 3 天
```

### 验证改进效果
```
在「改进中」表格更新「当前状态」
如果问题消失 → 移动到「已关闭」
如果问题持续 → 调整方案或升级优先级
```

### 每周审查（Heartbeat 时执行）
```
1. 检查「未解决问题」是否有增加
2. 检查「改进中」是否有过期（>7天无进展）
3. 检查指标趋势
4. 决定是否需要 Tier 2/3 调整
```
