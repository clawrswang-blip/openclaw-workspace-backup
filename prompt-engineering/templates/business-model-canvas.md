# business-model-canvas — 商业模型画布

## 触发条件
- 你需要梳理/评估一个商业模式
- 目标：找到商业模式的问题，不是"画个图"
- 场景：创业评估、项目复盘、商业计划

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| business_name | 业务名称 | Sungiven会员体系 |
| business_stage | 阶段 | 概念 / 早期 / 成熟 |
| canvas_focus | 重点 | 整体 / 某个模块 |
| known_blocks | 已知的模块 | 客户细分、成本结构 |
| unknown_blocks | 未知的模块 | 渠道、价值主张 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你画的商业模式画布，不是"九宫格填空"，是"发现你不知道你不知道"。

---

**业务：** {business_name}
**阶段：** {business_stage}
**重点模块：** {canvas_focus}
**已知模块：** {known_blocks}
**未知模块：** {unknown_blocks}

---

**第一步：毒辣之眼 — 这个商业模式最大的"定时炸弹"是什么？**

在动笔之前，先问：
- 哪个模块最脆弱？（一个模块崩了，整个模式就崩）
- 最依赖的假设是什么？（如果这个假设错了？）
- 现金流什么时候能转正？（这是创业者最容易自欺的）

**第二步：九宫格分析**

商业模式画布九宫格：

| 模块 | 问题 | 分析 |
|------|------|------|
| 客户细分 | 谁是你的客户？ | 已知/未知 |
| 价值主张 | 你解决什么问题？ | 已知/未知 |
| 渠道 | 怎么触达客户？ | 已知/未知 |
| 客户关系 | 怎么维护关系？ | 已知/未知 |
| 收入来源 | 怎么赚钱？ | 已知/未知 |
| 核心资源 | 什么资产是关键？ | 已知/未知 |
| 关键活动 | 做什么是必须的？ | 已知/未知 |
| 关键合作 | 谁是必须的？ | 已知/未知 |
| 成本结构 | 钱花在哪里？ | 已知/未知 |

**第三步：假设与验证**

每个模块都有假设：
- 哪些假设已经被验证？
- 哪些假设是"相信"而非"验证"？
- 验证成本有多高？

**第四步：闭环检查**

- 价值创造：这个模式真的在创造价值吗？
- 价值传递：客户怎么知道这个价值？
- 价值捕获：钱怎么流向你？可持续吗？

**第五步：输出结构**

```
[业务概览]
  {business_name} / {business_stage}

[核心洞察] — 一句话
  [这个模式最大的问题或机会]

[九宫格展开]
  [每个模块的详细分析]

[关键假设]
  假设列表
  验证状态

[风险分析]
  最大的3个风险
  应对方案

[优化建议]
  优先级排序
  短期/中期/长期
```

**第六步：自检清单**

- [ ] 每个模块都有内容吗？（不能空着）
- [ ] 模块之间有逻辑一致性吗？
- [ ] 关键假设被识别了吗？
- [ ] 风险有应对方案吗？
- [ ] 有"定时炸弹"识别吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【商业策略类·闭环强化】
□ 是否明确了「谁来买单、凭什么、持续多久」？
□ 风险点是否说清楚了？
□ 是否有清晰的行动边界？
□ 是否给出了「如果这样就失败」的预警？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
