# landing-page-copy — 营销落地页文案

## 触发条件
- 你需要一个落地页，能直接转化
- 目标：访问→行动（注册/购买/留言）
- 场景：产品发布、活动推广、线索收集

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| product_name | 产品/服务名称 | AI转型咨询服务 |
| main_benefit | 核心利益（1句话） | 30天让你的团队用上AI工作流 |
| target_audience | 目标人群 | 中小制造企业老板 |
| pain_point | 目标人群的痛点 | 知道AI有用但不知道怎么落地 |
| proof | 社会认同/证据 | 服务过XX家企业，客户反馈X% |
| cta | 行动召唤 | "预约咨询" / "获取方案" |
| urgency | 紧迫性（可选） | 限时优惠 / 名额有限 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的落地页文案，不是"介绍产品"，是"解决一个问题"。用户看完觉得"这就是我需要的"。

---

**产品：** {product_name}
**核心利益：** {main_benefit}
**目标人群：** {target_audience}
**痛点：** {pain_point}
**社会认同/证据：** {proof}
**行动召唤：** {cta}
**紧迫性（可选）：** {urgency}

---

**第一步：毒辣之眼 — 这个落地页最常见的失败是什么？**

在动笔之前，先问：
- 用户（{target_audience}）打开这个页面的第一秒在想什么？（"这和我有什么关系？"）
- 你最大的"不说"是什么？（优势不说自己说，让证据说话）
- 转化最大的障碍是什么？（价格？信任？需求不明确？）

**第二步：AIDA结构**

```
Attention（注意力）→ 钩子，痛点共鸣
Interest（兴趣）→ 展开，解决方案
 Desire（欲望）→ 欲望，让用户"想要"
 Action（行动）→ CTA，清晰的行动召唤
```

**第三步：信任建立**

不用自己说，用证据：
- {proof}
- 具体数字
- 真实案例
- 用户评价

**第四步：CTA设计**

{cta} 要：
- 具体（不是"联系我们"，是"预约30分钟免费咨询"）
- 低门槛（让人感觉"没有损失"）
- 明确（知道点击后会发生什么）

**第五步：输出结构**

```
[Hero区] — 一句话钩子
  不是"我们很厉害"，是"{target_audience}的{痛点}，我们来解决"

[问题区] — 痛点共鸣
  {pain_point}
  让用户说"这就是我"

[解决方案区] — 产品核心价值
  {main_benefit}
  不是功能列表，是解决方案

[证据区] — 社会认同
  {proof}

[紧迫性]（如有）
  {urgency}

[CTA区] — 行动召唤
  {cta}
```

**第六步：自检清单**

- [ ] 用户（{target_audience}）第一秒能看到"和我有关"吗？
- [ ] 痛点（{pain_point}）说准了吗？
- [ ] 解决方案（{main_benefit}）清晰吗？
- [ ] 证据（{proof}）够吗？真实吗？
- [ ] CTA（{cta}）具体、低门槛、明确吗？
- [ ] 紧迫性（{urgency}）真实吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【品牌/内容类·辨识度强化】
□ 是否有独特的叙事声音和视角？
□ 是否避免了所有品牌都适用的废话？
□ 是否有让人想截图的金句？
□ 情绪节奏是否符合目标读者的状态？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
