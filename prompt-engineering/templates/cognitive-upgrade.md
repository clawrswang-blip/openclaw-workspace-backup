# cognitive-upgrade — 个人认知升级

## 触发条件
- 你想升级自己的思维模型
- 目标：不是"知道更多"，是"想得更透"
- 场景：自我反思、读书笔记、认知迭代

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| current_belief | 现有认知/信念 | "努力就能成功" |
| new_input | 新输入 | 读到/学到的观点 |
| cognitive_conflict | 认知冲突点 | 努力方向比努力程度更重要 |
| upgrade_goal | 升级目标 | 建立更准确的因果模型 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你做的认知升级，不是"换一个新想法"，是"建立更准确的因果模型"。

---

**现有认知：** {current_belief}
**新输入：** {new_input}
**认知冲突点：** {cognitive_conflict}
**升级目标：** {upgrade_goal}

---

**第一步：毒辣之眼 — 你不愿改变的是什么？**

在动笔之前，先问：
- 你坚持{current_belief}多久了？它帮过你什么？（承认它的价值）
- 什么证据会让你改变这个信念？（先想清楚）
- 你最不想承认的是什么？（往往是最重要的）

**第二步：认知冲突分析**

{current_belief} vs {new_input}：

```
[{current_belief}]
  - 这个信念哪部分是对的？
  - 它在什么时候成立？
  - 它失败在哪里？

[{new_input}]
  - 这个观点哪部分是对的？
  - 它在什么时候成立？
  - 它的局限在哪里？

[冲突点]
  {cognitive_conflict}
  - 这是一个真冲突还是假冲突？
  - 两者可以并存吗？
```

**第三步：模型升级**

不是"非此即彼"：

**旧模型：** {current_belief}

**新模型：**[整合后的更准确的模型]

新模型要回答：
- 比旧模型多解释什么？
- 比旧模型少解释什么？
- 边界条件是什么？

**第四步：实践检验**

新模型怎么验证？
- 设计一个"最小验证实验"
- 什么结果会证明新模型是对的？
- 什么结果会证明新模型错了？

**第五步：十年视角**

这个认知升级在10年后的意义：
- 这个信念影响过你哪些重大决策？
- 新模型会如何改变你未来的决策？
- 如果不变，10年后会后悔吗？

**第六步：输出结构**

```
[认知主题]
  {upgrade_goal}

[核心洞察] — 一句话
  [这个认知升级最重要的点]

[现有信念]
  {current_belief}
  [它的价值]
  [它的局限]

[认知冲突]
  {new_input}
  {cognitive_conflict}

[模型升级]
  旧模型：{current_belief}
  新模型：[整合后的]

[验证设计]
  [最小实验]

[十年视角]
  [影响分析]
```

**第七步：自检清单**

- [ ] {current_belief}的价值被承认了吗？
- [ ] {new_input}的局限性被识别了吗？
- [ ] 新模型比旧模型更准确吗？
- [ ] 有验证方法吗？
- [ ] 10年后看，这个升级重要吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【个人成长类·真实挣扎强化】
□ 是否承认了困难和不确定性？
□ 是否有具体的试错过程？
□ 是否有「我以为...但其实...」的反转？
□ 是否给了「还没解决」的部分？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
