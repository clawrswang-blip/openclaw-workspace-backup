# xhs-flash-sale — 限时活动推广

## 触发条件
- 你有一个限时优惠/活动要推
- 目标：在有限时间内最大化转化
- 关键：紧迫感要真，不能狼来了

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| activity_name | 活动名称 | 母亲节有机礼盒 |
| discount | 优惠内容 | 全场8折 + 满100送30 |
| duration | 活动时间 | 5月10日-12日（共3天） |
| deadline_urgency | 紧迫性来源 | 限量500份 / 倒计时 |
| target_audience | 目标人群 | 温哥华送礼人群 |
| cta | 行动召唤 | "戳我下单" / "留言抢位" |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的小红书限时推广，不是催命符，是"帮受众占便宜"的机会感。

---

**活动名称：** {activity_name}
**优惠内容：** {discount}
**活动时间：** {duration}
**紧迫性来源：** {deadline_urgency}
**目标人群：** {target_audience}
**行动召唤：** {cta}

---

**第一步：毒辣之眼 — 你的优惠为什么"不值得抢"？**

在动笔之前，先问：
- 这个优惠的"真实价值"是什么？（不是标价，是受众感受到的）
- 受众最大的犹豫是什么？（价格？需求？还是"我不需要"？）
- 如果不说"限时限量"，受众凭什么现在就行动？

**紧迫感的真与假：**
- 真的紧迫：限量、真的售完为止、时间节点有意义
- 假的紧迫：天天"最后一天"、夸大其词的"史低价"

**第二步：价值叙事**

优惠不是"便宜"，是"值得"：
- 先说"这个东西本身值多少"
- 然后说"现在因为XXX原因，你可以用更少得到"
- 最后说"这个机会窗口是什么"

不要说"打折"，说"用X的价格，获得Y的价值"。

**第三步：紧迫性包装**

{deadline_urgency}是真实的吗？
- 真实：说清楚，让受众相信
- 夸大：宁可不说，也不要失信

**第四步：输出结构**

```
[开头钩子] — 一个具体的好处
  不是"打折了"，是"用半价拿到平时XXX的东西"

[活动核心] — 优惠内容
  {discount}
  说清楚：什么、多少钱、怎么拿

[为什么现在] — 紧迫性
  {deadline_urgency}

[目标人群画像] — 这个活动是给谁的
  {target_audience}
  不是所有人，是"你"

[行动召唤] — 具体、直接
  {cta}
  不要"感兴趣可私信"，要"现在就戳这里"
```

**第五步：自检清单**

- [ ] 价值叙事是"值得"还是"便宜"？
- [ ] 紧迫性（{deadline_urgency}）是真实的吗？
- [ ] 受众（{target_audience}）能对号入座吗？
- [ ] 行动召唤（{cta}）具体吗？可执行吗？
- [ ] 有没有"如果不抢就亏了"的合理感觉？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
