# wechat-longform — 公众号深度长文

## 触发条件
- 你需要一篇3000-5000字的深度长文
- 目标：建立专业信任 + 深度转化
- 平台：微信公众号（或其他长文平台）
- 关联：khazix-writer skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| topic | 文章主题 | 温哥华有机食品行业的真相 |
| thesis | 核心论点（1句话） | 不是所有"有机"都是真有机 |
| target_length | 目标字数 | 3000字 |
| cta | 行动召唤 | 关注公众号 / 加群 / 购买 |
| business_goal | 商业目标 | 引流 / 品牌建立 / 直接转化 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的公众号长文，不是"文章"，是"一场认知升级"。读者读完觉得"我以前想错了"或者"原来是这样"。

---

**主题：** {topic}
**核心论点：** {thesis}
**目标字数：** {target_length}
**行动召唤：** {cta}
**商业目标：** {business_goal}

---

**第一步：毒辣之眼 — 这篇文章存在的意义是什么？**

在动笔之前，先问：
- 这篇文章的"增量信息"是什么？（读者读完能得到什么他们不知道的？）
- 你（写手）最大的"认知盲区"是什么？（自以为是的地方）
- 受众最大的"认知误区"是什么？（他们以为对的，其实错的）
- 这篇文章发表后，读者会有什么实际行动？（不是"学到知识"，是具体行为改变）

**第二步：结构设计**

长文不是"流水账"，是"推理小说"：
- 每个章节要有"悬念"
- 每个章节要有"答案+新问题"
- 整体有"认知递进"

**第三步：叙事节奏**

```
引入（500字）→ 痛点共鸣
展开（2000字）→ 层层拆解
高潮（500字）→ 核心洞察
收尾（500字）→ 行动召唤
```

**第四步：闭环检查**

- 价值创造：这篇文章帮读者解决什么问题？
- 价值传递：用什么方式让读者相信你的判断？
- 价值捕获：你（品牌）从中得到什么？（{business_goal}）

**第五步：输出结构**

```
[开头钩子] — 直接进入核心矛盾
  不是"今天我想聊聊"，是一个具体案例/数据/反常识事实

[核心论点展开] — 3-5个分论点
  每个分论点：
  - 误区（先破）
  - 真相（再立）
  - 证据（数据/案例/逻辑）

[深度洞察] — 你的独家视角
  {thesis}
  不是搬运，是你的判断

[实操建议] — 可执行的结论
  读者读完应该做什么？

[结尾] — 情感+行动
  不是"感谢阅读"，是{cta}
```

**第六步：自检清单**

- [ ] 核心论点（{thesis}）够锐利吗？（不是废话）
- [ ] 有没有"认知增量"？（不是已知信息）
- [ ] 叙事节奏能让人读完吗？
- [ ] 证据链完整吗？（数据/案例支撑）
- [ ] 闭环（{business_goal}）能达到吗？
- [ ] 行动召唤（{cta}）清晰吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【品牌/内容类·辨识度强化】
□ 是否有独特的叙事声音和视角？
□ 是否避免了所有品牌都适用的废话？
□ 是否有让人想截图的金句？
□ 情绪节奏是否符合目标读者的状态？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
