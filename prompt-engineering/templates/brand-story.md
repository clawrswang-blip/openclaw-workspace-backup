# brand-story — 品牌故事/branding

## 触发条件
- 你需要完整的品牌故事（官网/介绍页/提案用）
- 目标：让人"记住你是谁、为什么存在"
- 不是产品说明书，是信念宣言

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| brand_name | 品牌名称 | Sungiven Foods |
| origin | 创世故事 | 创始人背景、为什么做这件事 |
| mission | 使命 | 让海外华人吃到家的味道 |
| vision | 愿景（3-5年） | 北美亚裔社区的信任之选 |
| values | 核心价值观（1-3个） | 真实、新鲜、连接 |
| tone | 风格 | 真诚朴素 / 有调性 / 有情怀 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的品牌故事，不是"我们成立于X年，提供Y服务"——是"我们为什么存在"。

---

**品牌：** {brand_name}
**创世故事：** {origin}
**使命：** {mission}
**愿景：** {vision}
**价值观：** {values}
**风格：** {tone}

---

**第一步：毒辣之眼 — 这个品牌最怕什么？**

在动笔之前，先问：
- 这个品牌最大的"弱点"是什么？（不是产品，是信念层面的）
- 受众选择这个品牌的真正原因是什么？（不是功能，是情感）
- 品牌故事的"对立面"是什么？（不是什么，确立了是什么）

**第二步：故事结构**

好的品牌故事有3层：
1. **过去** — 为什么存在（origin）
2. **现在** — 正在做什么（mission）
3. **未来** — 要成为什么（vision）

**第三步：品牌人格**

{brand_name}是谁？
- 像一个人，不是"企业"
- 有态度，不讨好所有人
- 有温度，不是冷冰冰的服务

**第四步：输出结构**

```
[品牌标题] — 一句话宣言
  不是"X品牌介绍"，是一个态度

[创世故事] — 为什么存在
  {origin}
  要有：真实的困难、真实的犹豫、真实的转折

[使命宣言] — 我们在做什么
  {mission}
  不是口号，是可衡量的承诺

[愿景描述] — 我们要成为什么
  {vision}

[价值观] — 我们怎么做事
  {values}
  每个价值观要有故事/案例支撑

[结尾] — 一句话信念
  回应开头，形成闭环
```

**第五步：自检清单**

- [ ] 有"真实的困难"吗？（不是一帆风顺）
- [ ] 受众能产生情感共鸣吗？
- [ ] 是"记住这个品牌"还是"觉得这个品牌不错"？
- [ ] 风格（{tone}）统一吗？
- [ ] 价值观（{values}）有支撑吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【品牌/内容类·辨识度强化】
□ 是否有独特的叙事声音和视角？
□ 是否避免了所有品牌都适用的废话？
□ 是否有让人想截图的金句？
□ 情绪节奏是否符合目标读者的状态？

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
