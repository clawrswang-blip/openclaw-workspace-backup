# xhs-scenario-content — 场景化内容创作

## 触发条件
- 你想做一个"场景化"的内容，让用户"看到自己"
- 目标：不是展示产品，是展示"使用场景"
- 核心：让受众说"这就是我的日常"

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| scenario | 场景 | 二娃妈妈的周末早餐 |
| product_mention | 产品植入方式 | 自然的 / 软性 / 硬性 |
| emotion | 场景情绪 | 温馨 / 治愈 / 真实 |
| narrative_pov | 叙事视角 | 第一人称 / 第三人称观察 |
| goal | 内容目标 | 种草 / 建立形象 / 引流 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的小红书场景内容，不是"产品说明书"，是"生活切片的艺术"。

---

**场景：** {scenario}
**产品植入方式：** {product_mention}
**场景情绪：** {emotion}
**叙事视角：** {narrative_pov}
**内容目标：** {goal}

---

**第一步：毒辣之眼 — 这个场景最大的"假"是什么？**

在动笔之前，先问：
- 大多数品牌做的场景内容，"假"在哪里？（太精致？太完美？）
- 真实的生活场景是什么样的？（有混乱、有妥协、有意外）
- 这个场景里，用户真正需要的是什么？（不是产品，是"被理解"）

**第二步：场景三要素**

好的场景内容有：
1. **具体的时空坐标** — 不是"早上"，是"周一早上7:15，厨房里还有蒜味"
2. **真实的情绪流动** — 不是"幸福的一天"，是"我本来想放弃，后来..."
3. **自然的解决方案** — 不是"用了X产品，真好"，是"这种情况，只能是这样"

**第三步：产品植入的分寸**

{product_mention}：
- 自然的：产品是场景的一部分，不突兀
- 软性：提到，但故事核心不是产品
- 硬性：明确推荐，但用"这种情况，我选它"的语气

**第四步：输出结构**

```
[开头] — 一个具体的时空切片
  {narrative_pov}
  不是"周末早上"，是"周六早上9点，阳光刚好..."

[场景展开] — 情绪+情节
  遇到什么问题？
  心情是什么？
  怎么解决的？（产品在这里自然出现）

[情绪高点] — 场景的情绪核心
  {emotion}
  不是"产品很好用"，是"这种感觉..."

[留白] — 不说完
  结尾不是总结，是一个画面
  让读者自己代入
```

**第五步：自检清单**

- [ ] 场景足够具体吗？（时间、地点、情绪都在）
- [ ] 情绪（{emotion}）精准吗？
- [ ] 产品植入（{product_mention}）自然吗？
- [ ] 受众能"看到自己"吗？
- [ ] 目标（{goal}）达到了吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【XHS类·真实感强化】
□ 是否像真实用户的分享，而不是软文？
□ 是否有「踩坑」「后悔」「惊喜」这类真实情绪？
□ 是否有只有真实用过才知道的细节？
□ 图片描述是否符合小红书风格的视觉语言？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
