# market-analysis-cn — 市场分析-中国

## 触发条件
- 你需要分析中国市场（消费者、行业、竞争）
- 目标：找到机会，不是泛泛的市场规模数字
- 关联：market-analysis-cn skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| market_scope | 市场范围 | 社区团购 / 直播电商 / 有机食品 |
| region | 区域 | 中国二三线城市 |
| analysis_type | 分析类型 | 消费者洞察 / 竞争格局 / 机会评估 |
| goal | 目的 | 评估是否进入中国市场 |
| timeline | 时间维度 | 现状 / 趋势 / 预测 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你做的中国市场分析，不是"数据堆砌"，是"穿透噪音，找到真相"。

**第一步：毒辣之眼 — 你对中国市场最大的误解是什么？**

在动笔之前，先问：
- 大多数人对{market_scope}市场的误解是什么？
- 真正的机会不在"热门"，在"边缘"——在哪里？
- 外资/外来者最大的"不知道自己不知道"是什么？

**第二步：中国市场特殊因素**

中国市场分析必须考虑：
- 监管环境（政策风险）
- 平台生态（微信/抖音/小红书，不同逻辑）
- 消费者行为差异（不是"中国消费者"，是"XX类型消费者"）
- 竞争格局（本土玩家 vs 外来者）

**第三步：闭环框架**

- 价值创造：谁能用这个分析做出正确决策？
- 价值传递：分析能被正确理解吗？（避免误读）
- 价值捕获：谁会为这个分析付费？（商业逻辑）

**第四步：输出结构**

```
[市场概览] — 规模、增速、格局
  关键数字（不是全部数字）
  数据来源

[核心洞察] — 3-5条
  每条：
  - 发现
  - 证据
  - 对{goal}的意义

[{analysis_type}详细分析]
  [根据类型展开]

[机会与风险]
  机会：
  - 优先级

  风险：
  - 概率
  - 影响

[结论与建议]
  {goal}的核心结论
  具体建议
```

**第五步：自检清单**

- [ ] 有没有"穿透数据看到本质"？
- [ ] 监管/政策风险评估了吗？
- [ ] 竞争格局有本土视角吗？
- [ ] 对{goal}有直接指导意义吗？
- [ ] 结论不是废话吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【深度分析类·穿透力强化】
□ 是否回答了「为什么会这样？」而不是「是这样的」？
□ 是否有第二层、第三层因果分析？
□ 是否提出了反直觉的洞察？
□ 是否明确了「谁会不同意这个结论」？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
