# seo-article — SEO 文章

## 触发条件
- 你需要一篇能"搜到"的文章
- 目标：搜索排名 → 自然流量
- 关联：ai-seo skill, seo-audit skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| target_keyword | 目标关键词 | 温哥华有机食品外送 |
| article_angle | 文章角度 | 怎么选、为什么、多少钱 |
| target_length | 目标字数 | 2000字 |
| competitor_pages | 竞品页面（可选） | 链接1、链接2 |
| tone | 风格 | 专业测评 / 实用指南 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的SEO文章，不是"塞关键词"，是"真正回答问题"。Google喜欢，用户更喜欢。

**第一步：毒辣之眼 — 搜索这个词的人真正想知道什么？**

在动笔之前，先问：
- 搜索"{target_keyword}"的人，在找什么？（意图分析）
- 竞品页面（{competitor_pages}）的"没说"是什么？（差异化机会）
- Google/搜索引擎为什么应该把这个页面排上去？（E-E-A-T）

**第二步：意图分析**

搜索意图有3种：
1. **信息型** — 想知道什么（how/what/why）
2. **导航型** — 想去哪里（品牌名）
3. **交易型** — 想做什么（buy/download/sign up）

{article_angle}对应的意图是什么？

**第三步：内容结构（SEO标准格式）**

```
[Title Tag] — 60字符内，包含关键词
[H1] — 和Title一致
[Meta Description] — 155字符内，摘要+CTA

[Introduction] — 钩子+承诺
  搜索这个词的人能得到什么

[目录/大纲] — 结构清晰
  H2/H3层级

[主体内容] — 逐一展开
  每个section要有：
  - 关键词自然出现
  - 具体信息（不是废话）
  - 数据/案例支撑

[FAQ] — 常见问题
  搜索这个词的人还会问什么

[Conclusion] — 总结+CTA
```

**第四步：关键词布局**

- Title：1次
- H1：1次
- H2：2-4次
- 正文：自然出现（密度1-2%）
- URL：简短、含关键词

**第五步：自检清单**

- [ ] 意图（{article_angle}）分析准确吗？
- [ ] Title和H1包含关键词了吗？
- [ ] 内容比竞品（{competitor_pages}）更有价值吗？
- [ ] 结构有FAQ/目录吗？
- [ ] 关键词密度自然吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【品牌/内容类·辨识度强化】
□ 是否有独特的叙事声音和视角？
□ 是否避免了所有品牌都适用的废话？
□ 是否有让人想截图的金句？
□ 情绪节奏是否符合目标读者的状态？

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
