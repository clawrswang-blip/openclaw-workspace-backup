# launch-strategy — Launch Strategy

## 触发条件
- 你有一个产品/服务要发布
- 目标：最大化首发声量 → 首批用户
- 场景：新品发布、项目启动、内部上线

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| product_name | 产品名称 | Sungiven App 2.0 |
| launch_type | 发布类型 | 公开上线 / 限量内测 / 邀请制 |
| launch_date | 上线日期 | 2026年6月1日 |
| target_users | 目标用户 | 温哥华现有会员 |
| key_message | 核心信息 | 全新会员电商体验 |
| success_metric | 成功指标 | 500注册 / 100购买 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你的Launch Strategy，不是"发个公告"，是"一个事件"。

**第一步：毒辣之眼 — 你的Launch最可能失败在哪里？**

在动笔之前，先问：
- 最大的"Launch失败"是什么？（没人在意？期待管理失败？执行出问题？）
- 目标用户（{target_users}）为什么会传播这个消息？（不是"因为好"，是因为"和我有关"）
- 在{launch_date}之前，最可能出问题的环节是什么？

**第二步：Launch准备检查清单**

```
[认知阶段]（Launch前2-4周）
  - 核心信息确定
  - 预热内容准备
  - KOL/种子用户沟通

[爆发阶段]（Launch日）
  - 官方发布
  - 传播素材
  - 应急响应

[延续阶段]（Launch后1-2周）
  - 用户反馈收集
  - 内容续热
  - 数据复盘
```

**第三步：传播设计**

为什么{target_users}会传播{product_name}？
- 要有"分享动机"（不是"帮我宣传"，是"我想让朋友知道"）
- 要有"分享素材"（图/文案/短视频）
- 要有"分享时机"（Launch日是事件，不是日常）

**第四步：风险预案**

```
[最大风险1] — 预案
[最大风险2] — 预案
[最大风险3] — 预案
```

**第五步：输出结构**

```
[Launch概览]
  {product_name} / {launch_date} / {success_metric}

[核心洞察] — 一句话
  [Launch真正要解决的问题]

[Launch Timeline]
  [详细时间表]

[传播计划]
  - 预热
  - 爆发
  - 延续

[风险预案]
  [风险+应对]

[成功指标追踪]
  [{success_metric}]
```

**第六步：自检清单**

- [ ] 有没有"Launch事件感"？（不是"发个通知"）
- [ ] 传播动机设计了吗？
- [ ] 风险预案有了吗？
- [ ] 成功指标（{success_metric}）可衡量吗？
- [ ] Timeline可行吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【商业策略类·闭环强化】
□ 是否明确了「谁来买单、凭什么、持续多久」？
□ 风险点是否说清楚了？
□ 是否有清晰的行动边界？

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
