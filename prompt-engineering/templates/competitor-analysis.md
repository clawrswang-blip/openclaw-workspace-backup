# competitor-analysis — 竞品分析

## 触发条件
- 你需要了解竞争对手
- 目标：找到机会点，不是做"表面研究"
- 关联：competitor-alternatives skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| your_product | 你的产品/服务 | Sungiven会员体系 |
| competitors | 竞品列表 | 竞品A、竞品B、竞品C |
| analysis_dimension | 分析维度 | 产品/价格/用户体验/营销 |
| goal | 分析目的 | 找到差异化机会 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你做的竞品分析，不是"他们有什么"，是"他们做不到什么"。

---

**你的产品：** {your_product}
**竞品列表：** {competitors}
**分析维度：** {analysis_dimension}
**分析目的：** {goal}

---

**第一步：毒辣之眼 — 你没看竞品的什么？**

在动笔之前，先问：
- 竞品最大的"弱点"是什么？（不是产品缺陷，是"他们选择不做什么"）
- 竞品最怕什么？（你做什么会让他们难受）
- {goal}的真正机会在哪里？（不是"比他们好"，是"他们没做的"）

**第二步：分析框架**

```
[竞品概览] — 基础信息
  产品、定位、目标人群、规模

[维度对比] — 逐一分析
  {analysis_dimension}
  每个维度：
  - 竞品怎么做
  - 你的怎么做
  - 差异点

[竞品弱点] — 机会识别
  - 他们选择不做的地方（为什么？）
  - 他们的用户不满是什么？（差评分析）
  - 他们的盲区是什么？（市场变化他们没跟上）

[机会矩阵] — 行动指南
  | 机会 | 竞品现状 | 你的能力 | 优先级 |
```

**第三步：闭环检查**

- 价值创造：这个分析能帮你改善{your_product}吗？
- 价值传递：你能把这个洞察传达给团队吗？
- 价值捕获：这个分析能带来什么竞争优势？

**第四步：输出结构**

```
[竞品概览]
  竞品A / 竞品B / 竞品C

[核心发现] — 一句话总结
  [毒辣洞察]

[详细对比]
  维度1
  维度2
  ...

[机会点]
  优先级排序

[行动建议]
  短期（1-3个月）
  中期（3-6个月）
```

**第五步：自检清单**

- [ ] 分析的是"他们做不到什么"还是"他们做了什么"？
- [ ] 有没有竞品弱点分析？
- [ ] 机会点有优先级吗？
- [ ] 行动建议具体吗？（不是"加强差异化"这种废话）

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【商业策略类·闭环强化】
□ 是否明确了「谁来买单、凭什么、持续多久」？
□ 风险点是否说清楚了？
□ 是否有清晰的行动边界？
□ 是否给出了「如果这样就失败」的预警？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
