# xhs-comparison — 测评对比文

## 触发条件
- 你想做"哪个更好"的对比
- 目标：帮受众做决策，同时建立专业信任
- 关键：不是站队，是帮读者看清楚

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| item_a | 对比项A | 有机蓝莓A品牌 |
| item_b | 对比项B | 有机蓝莓B品牌 |
| comparison_dimension | 对比维度 | 新鲜度/价格/口感/包装 |
| winner | 结论（有预设吗） | 有 / 无 |
| target_audience | 受众 | 注重健康的妈妈们 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你写的小红书测评对比，不是"收了钱就说好"，是用专业帮读者做理性的最优解。

---

**对比项A：** {item_a}
**对比项B：** {item_b}
**对比维度：** {comparison_dimension}
**预设结论：** {winner}
**目标受众：** {target_audience}

---

**第一步：毒辣之眼 — 你没说的是什么？**

在动笔之前，先问：
- 这个对比最大的"坑"是什么？（维度的选择本身就带偏见）
- 受众（{target_audience}）真正关心的维度是什么？不是你以为的。
- 两个对比项各自的"隐藏优势"是什么？

**第二步：维度设计原则**

对比维度要：
- 选对受众真正在意的（不是行业标准的"专业维度"）
- 维度之间不能有包含关系（"价格"和"性价比"是不同维度）
- 每个维度都能用具体事实支撑

**第三步：对比叙事**

对比不是表格，是故事：
- 先说"两个都很不错，但..."
- 每个维度说清楚"差异在哪里，为什么重要"
- 最后给出"如果你XXX，就选A；如果你XXX，就选B"

不要：
- "各有千秋，都很好"（废话）
- "X全面碾压Y"（太假）
- 只比"纸面参数"（没用）

**第四步：输出结构**

```
[开头钩子] — 一个真实的选择困境
  "我纠结了一个月，最后发现..."
  或者："99%的人选错了，只因为没看这个维度"

[两个选项简介] — 30秒背景
  {item_a} vs {item_b}

[维度对比] — 3-5个维度
  每个维度：
  - 结论（谁更好）
  - 证据（具体事实）
  - 对受众意味着什么

[选购建议] — 不含糊
  "如果你重视XXX，选A；如果你重视XXX，选B"

[无预设结论时] — 如实说
  "这次对比，我没有找到明显赢家，以下是我的分析..."
```

**第五步：自检清单**

- [ ] 维度选择对受众（{target_audience}）有意义吗？
- [ ] 有没有"帮读者做决策"而不是"替读者做决定"？
- [ ] 证据是具体事实还是形容词？
- [ ] 结论（{winner}）有说服力吗？或者说清楚为什么"没有赢家"了吗？
- [ ] 有没有帮读者找到"适合自己的"而不是"最好的"？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【XHS类·真实感强化】
□ 是否像真实用户的分享，而不是软文？
□ 是否有「踩坑」「后悔」「惊喜」这类真实情绪？
□ 是否有只有真实用过才知道的细节？
□ 图片描述是否符合小红书风格的视觉语言？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
