# infographic-design — 信息图设计

## 触发条件
- 你需要一张信息图
- 目标：让复杂信息一目了然
- 关联：baoyu-infographic skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| infographic_topic | 主题 | 温哥华有机食品消费趋势 |
| data_points | 数据点（3-7个） | 数据1、数据2 |
| key_message | 核心信息（1句话） | 亚裔社区有机食品消费增速最快 |
| visual_style | 视觉风格 | 简约数据风 / 插画风 / 地图叠加 |
| output_format | 输出格式 | 长图 / 方形 / A4 |
| platform | 使用平台 | XHS / 公众号 / 官网 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你设计的信息图，不是"数据可视化"，是"让人一眼记住一个结论"。

**输出格式：** {output_format}

**第一步：毒辣之眼 — 信息图最大的失败是什么？**

在动笔之前，先问：
- 受众看完这张图，能说出一个结论吗？（不能就是失败）
- 最大的问题是什么？"数据堆砌"还是"没有灵魂"？
- 这张图的核心价值是什么？（不是"展示数据"，是"传递洞察"）

**第二步：信息架构**

{key_message} 是核心——一切围绕它：

```
[标题/结论] — 一句话说清楚
  {key_message}

[支撑数据] — {data_points}
  每个数据点：
  - 数字/事实
  - 一句话解读

[数据来源]（如需）
  小字说明

[行动召唤/品牌]（如需）
  CTA或品牌名
```

**第三步：视觉设计原则**

{visual_style} 的选择依据：
- 简约数据风：适合专业受众、数字敏感
- 插画风：适合大众受众、传播为主
- 地图叠加：适合地域相关数据

设计原则：
- 一张图只传递一个核心信息
- 数据密度要适中（太多=没重点）
- 留白是呼吸，不是浪费

**第四步：平台适配**

{platform} 决定：
- 尺寸：{output_format}
- 信息量：社交媒体<官网
- 行动召唤：需要/不需要

**第五步：输出结构**

```
[信息图设计 Brief]

### 核心信息
{key_message}

### 信息架构
[层级结构]

### 数据描述
每个数据点的说明

### 视觉风格
{visual_style}
配色建议
字体建议

### 输出规格
{output_format}
{platform}
```

**第五步：自检清单**

- [ ] 受众能一眼说出核心结论（{key_message}）吗？
- [ ] 数据点（{data_points}）都在支撑核心结论吗？
- [ ] 视觉风格（{visual_style}）适合受众和平台吗？
- [ ] 数据密度适中吗？
- [ ] {platform} 适配了吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【品牌/内容类·辨识度强化】
□ 是否有独特的叙事声音和视角？
□ 是否避免了所有品牌都适用的废话？
□ 是否有让人想截图的金句？
□ 情绪节奏是否符合目标读者的状态？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
