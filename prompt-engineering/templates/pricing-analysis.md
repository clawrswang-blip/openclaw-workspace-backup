# pricing-analysis — 定价策略分析

## 触发条件
- 你需要制定/调整定价策略
- 目标：最大化利润 or 最大化市场份额（这两个经常矛盾）
- 关联：pricing-strategy skill

## 输入变量

| 变量 | 说明 | 示例 |
|------|------|------|
| product_service | 产品/服务 | 会员年卡 |
| current_pricing | 当前定价 | C$199/年 |
| pricing_objective | 定价目标 | 提升客单价 |
| market_position | 市场定位 | 中高端 |
| competitor_pricing | 竞品定价 | A品牌C$149、B品牌C$299 |
| cost_structure | 成本结构（可选） | 已知/未知 |

## 核心 Prompt

===新增===
【质量锚点：Top 3% 标准】
- 不是「够用」，是「让人想收藏」
- 不是「说明白了」，是「看完觉得值」
- 不是「通用答案」，是「这个细分领域的深层洞察」
- 不是「表面总结」，是「别人没看到的维度」
- 所有数据必须有来源，所有观点必须有依据
===新增===

你做的定价分析，不是"成本+利润=定价"，是"价格背后的心理博弈"。

**第一步：毒辣之眼 — 定价最大的误区是什么？**

在动笔之前，先问：
- "贵"还是"便宜"是谁决定的？（不是成本，是顾客的感知价值）
- 定价目标（{pricing_objective}）和定价策略矛盾吗？（要市场份额还是要利润）
- 竞品定价（{competitor_pricing}）背后的逻辑是什么？（不是简单比价）

**第二步：价值锚点分析**

定价不是绝对的，是相对的：
- 锚点1：{competitor_pricing}的竞品A（低端）和B（高端）
- 锚点2：你的{product_service}放在哪里？
- 锚点3：有没有"诱饵套餐"？（引导选择）

**第三步：价格弹性**

- 提价10%，需求下降多少？
- 降价10%，需求上升多少？
- 哪个价格点是最优解？

**第四步：定价策略选项**

| 策略 | 逻辑 | 风险 |
|------|------|------|
| 渗透定价 | 低价切入，快速占领市场 | 未来涨价难 |
| 价值定价 | 基于感知价值定价 | 需要持续价值传递 |
| 竞争定价 | 跟随或低于竞品 | 价格战风险 |
| 动态定价 | 根据需求实时调整 | 用户信任风险 |

**第五步：闭环验证**

- 价值创造：这个定价能支撑产品持续提供价值吗？
- 价值传递：顾客能理解这个价格背后的价值吗？
- 价值捕获：这个定价下，利润结构是什么？

**第六步：输出结构**

```
[定价概览]
  {product_service} / {current_pricing}

[核心洞察] — 一句话
  [定价最大的机会或风险]

[市场定位分析]
  {market_position}
  竞品价格带

[价值锚点]
  锚点设计

[定价策略建议]
  方案A
  方案B
  推荐

[风险评估]
  风险1：...
  风险2：...

[实施建议]
  短期（调价方案）
  中期（价值传递优化）
```

**第六步：自检清单**

- [ ] 定价目标（{pricing_objective}）明确了吗？
- [ ] 价值锚点设计了吗？
- [ ] 价格弹性评估了吗？
- [ ] 风险有应对方案吗？
- [ ] 方案之间有清晰的取舍逻辑吗？

===新增===
【去AI化检查】
□ 是否有具体的个人观察/数据点，而不是泛泛而谈？
□ 是否有不完美的、真实的细节？（AI总爱给完美答案）
□ 句式是否有变化，还是全是「首先...其次...最后...」？
□ 是否有反直觉的洞见，而不是安全的主流观点？
□ 是否避免了「不仅...而且...」「随着...」这类AI最爱用的句式？
□ 是否有情绪温度，还是全程冷静得像产品说明书？
□ 是否有具体的场景/案例，而不是笼统的「比如说...」？
===新增===

===新增===
【深度分析类·穿透力强化】
□ 是否回答了「为什么会这样？」而不是「是这样的」？
□ 是否有第二层、第三层因果分析？
□ 是否提出了反直觉的洞察？
□ 是否明确了「谁会不同意这个结论」？
===新增===

## 自我迭代检查（必做）

### 产出后自检（Before 输出）
- [ ] 我提供的洞见，读者在其他地方能轻易找到吗？（如果是，降级重写）
- [ ] 我的核心观点能用一句话概括吗？（如果不能，结构有问题）
- [ ] 最有争议的那个点，我有没有回避？（回避了=不够真诚）

### 产出后迭代（After 初稿）
问题 1：标题是否足够吸引，让 80% 的人想点进来？
问题 2：开头是否在 3 秒内抛出「钩子」？
问题 3：是否有至少一个「反直觉」的洞察？
问题 4：结尾是否给出了行动指引或思考延伸，而不是「以上就是...」？
问题 5：是否符合该平台的语言风格？（XHS vs 公众号 vs 知乎 vs 微博）
