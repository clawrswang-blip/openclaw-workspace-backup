## 推送节奏

- **频率**：每周二、五推送（工作日早上 9 点 PT）
- **格式**：
  ```
  🔍 [方向] — [日期]
  
  看到了什么：
  1. [结论] — [reference: URL]
  2. [结论] — [reference: URL]
  3. [结论] — [reference: URL]
  
  对你意味着什么：
  ...
  
  要不要动：[是/否] — [理由]
  
  References:
  [1] ...
  [2] ...
  ```
- **数据要求**：必须使用 2025Q4 或 2026 年数据，标注数据来源和发布时间
- **每次来源**：2-3 个源头，每个方向 3 条洞见，每条必须附 reference
